FREE-GIGS WEBSITE
=================


Description:
------------
Free-Gigs is a web application for managing and attending free concerts. It allows users to register, view upcoming concerts, and book their attendance. Administrators can manage bands, venues, and concerts.

Features:
---------
- User registration and login
- Admin login and management
- View upcoming concerts
- Book concert attendance
- Manage bands, venues, and concerts (admin only)

Setup:
------
1. Set up a local web server on XAMPP, WAMP
2. Import the database schema included in MySQL
3. Place all files in your web server htdocs

Technologies Used:
------------------
- PHP
- MySQL
- HTML
- CSS
- JavaScript
